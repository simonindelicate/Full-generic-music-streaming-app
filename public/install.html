Set up your music site the simple way

No jargon. No bouncing between ten tools. Follow these cards in order. You can choose either: Simple mode (JSON file on Netlify, easiest) or Full mode (MongoDB + built-in upload workflow).

Tip: keep this page open and tick each box as you complete it.

1) Deploy to Netlify

Start here

In Netlify click Add new site → Import an existing project, choose this repo, then deploy.

Leave build settings as default unless you already changed them on purpose.

Netlify dashboard → Add new site → Import an existing project → Deploy site
After deploy, open your Netlify site URL.

2) Pick your data mode

Required

Simple mode (recommended for beginners): keep your tracks in a JSON file inside the site.

Full mode: use MongoDB when you want dynamic editing, API writes, and scaling.

So yes: MongoDB is optional for a basic install, but required for full admin/API features.

Simple mode file: public/albumfooter.json
Full mode uses MONGODB_URI + MONGODB_DB_NAME

3) Configure environment values (Full mode only)

Skip in Simple mode

If you chose Full mode, paste your MongoDB details below and copy the generated variables into Netlify.

If you chose Simple mode, leave this section and continue to step 4.

MONGODB_URI=
MONGODB_DB_NAME=
APP_BASE_URL=https://your-site.netlify.app
MEDIA_BASE_URL=https://media.yourdomain.com/music
FTP_HOST=
FTP_USER=
FTP_PASSWORD=
FTP_PUBLIC_BASE_URL=https://media.yourdomain.com
FTP_BASE_PATH=uploads
FTP_SECURE=false
ADMIN_PIN=1310
Paste these into Netlify → Site configuration → Environment variables.

4) Set up one-click uploads to external media hosting

Important

You were right: upload should happen inside this app, not by sending people away to another site.

Configure your storage credentials (S3-compatible or SFTP) as Netlify environment variables so the upload button can send files directly there and save the returned URL automatically.

Goal flow: Click upload in app → function sends to your media host → URL saved automatically
No manual "upload elsewhere then paste URL" workflow.

The upload buttons in `insert.html` already use `/.netlify/functions/uploadMedia`; once FTP environment variables are set, uploads happen in-app in one click.

5) Redeploy and verify

Go live
Netlify dashboard → Trigger deploy → Deploy site
Then open /player.html on your live URL
Example: https://your-site.netlify.app/player.html.
